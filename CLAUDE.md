# Building blocks for Web Site
In the SOUNDS folder, there are a number of MP3 files in subfolders which are named according to the language of the spoken text in the recording.

In the LANGUAGE-DATA folder, there are a number of JSON files. The name of each corresponds to the language of the data in each file. Each JSON file is an array of objects. Each object has four keys: soundNum, sound, text, and synonym.
